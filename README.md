# ControlServerBot

Простой бот для управления сервером.

## Установка

```bash
npm install
```

## Использование

```bash
node index.js
```

## Описание

Этот проект предназначен для удалённого управления сервером с помощью бота.  
Перед использованием убедитесь, что установлены все зависимости.
